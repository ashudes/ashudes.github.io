Unit 9: Developing an API for a Distributed Environment

WHAT?

Unit 9 followed on the practical exercise on API from Unit 7. It provided practical Python development skills, focusing on creating an API and experimenting with CRUD operations.

SO WHAT?

The unit's hands-on approach to API development and the exploration of Python's flask library are instrumental in understanding how back-end code developments can be accessed through a user interface.

NOW WHAT?

This was the first experience I had practicing with API development. It has made me eager to explore additional API development approaches in a distributed environment while also learning potential areas of vulnerability.
