Unit 5: More on Classes

Unit 5 delves deeper into the concept of classes in Python, focusing on assigning variable values using constructors and the use of abstract classes. The unit also explores the modularity of Python and how it is facilitated using inheritance, aggregation, and composition. It further examines the role of packages and namespaces in variable naming.

Learning Outcomes

By the end of the unit, students will be able to create a Python program using a constructor and an abstract class, describe the concepts of polymorphism, aggregation, and composition, and name variables according to their namespace.

Practical Work

The practical work in this unit will help students understand how to extend the development of a Python program using a constructor and an abstract class, which is crucial for the coding contribution of the summative assessment.
