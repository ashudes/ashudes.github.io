Summary:

Unit 9: Packaging and Testing

This Unit focuses on the packaging and testing of Python programs. It is designed to provide students with a comprehensive understanding of how to prepare a Python program for distribution and how to ensure its quality through testing.

Learning Outcomes:

Upon completion of this unit, students will be able to:

  • Understand and apply the concept of packaging a Python program.
  • Use the PEP257 convention to effectively document their Python code.
  • Understand and describe various approaches used to test code.
  • Use Python’s tools to test the quality of their code.

The knowledge and skills gained from this unit will be particularly useful in finalizing the code developed for the summative assessment, ensuring its quality and readiness for distribution. This unit represents one of the final steps in the Python development process, emphasizing the importance of code quality and documentation.
